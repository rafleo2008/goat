Read
```json
{
    "table_name": "pois",
    "mode": "read",
    "scenario_id": 10
}
```
Insert
```json
{
    "table_name": "pois",
    "mode": "insert",
    "features": [

        {
            "name": "Test1",
            "amenity": "tram_stop",
            "geom": "POINT(11.4043 48.1232)",
            "scenario_id": 10
        },
        {
            "name": "Test2",
            "amenity": "bus_stop",
            "geom": "POINT(11.4743 48.1332)",
            "scenario_id": 10
        }                
    ]

}

```

```json
{

        {
```
